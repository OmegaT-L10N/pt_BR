Grupo de usuários

OmegaT has a very active, supportive and multilingual user group. This is the recommended support channel, which users are strongly encouraged to join.


Listas de discussão

OmegaT também possui as seguintes listas de discussão e rastreadores, que estão hospedados no Sourceforge

OmTdev Lista de discussão dos desenvolvedores OmegaT. Esta lista tem como objetivo discutir tecnicamente funcionalidades e o desenvolvimento, é destinada a desenvolvedores do OmegaT e outras partes interessadas.
OmT-l10n Esta lista de discussão é voltada ao traodutores que trabalham na localização do OmegaT.

Sistema de rastreamento

Erros, solicitações de novas funcionalidades e documentação de problemas são tratados pelo Sourceforge através dos sistemas de rastreamento abaixo: Por favor, não use os sistemas de rastreamento antes de discutir o problema no grupo de usuários e/ou nas listas de discussão.
