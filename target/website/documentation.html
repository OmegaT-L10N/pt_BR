Manual do Usuário

O OmegaT é fornecido com um manual do usuário abrangente. Para abrir o manual a partir do OmegaT, simplesmente selecione Ajuda > Manual do usuário, ou pressione F1.

Um guia de início rápido, que traz orientações sobre os principais procedimentos e funções do OmegaT, é exibido na interface do usuário do OmegaT por padrão.

Também é possível abrir o manual do usuário fornecido com o OmegaT sem iniciar o programa. Ele pode ser encontrado na pasta "docs", na pasta de instalação do OmegaT.
