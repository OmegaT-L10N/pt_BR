A filosofia do código aberto

O OmegaT é um aplicativo de código aberto e de projetos. Isso significa que o código-fonte do OmegaT está disponível gratuitamente e que a licença do OmegaT proporciona aos seus usuários uma liberdade considerável na sua utilização. Não somente pode ser usada gratuitamente, mas os usuários também são livres para fazer cópias e distribuí-las.

Um aspecto importante dessa liberdade é que os usuários também podem modificar o código do programa. Para o movimento de código aberto, a liberdade que os usuários têm para alterar o programa a fim de atender às suas próprias necessidades é mais importante do que o fato de não precisarem pagar por ele. Isto é descrito como "free as in speech", em contraste com o "free as in beer".


A filosofia do projeto OmegaT

A mesma correlação pode ser encontrada no projeto OmegaT. Os usuários são livres para baixar e usar o programa. Por sua vez, são livres, e até mesmo encorajados (mas de nenhuma maneira obrigados) a contribuir com o projeto. Como decidem fazê-lo vai de acordo com o julgamento de cada um. Por exemplo, não há nada que impeça um usuário de escrever seu próprio manual de usuário ou um vídeo apresentando o OmegaT, e caso o projeto considere tal iniciativa útil, é provável que seja disponibilizado para a comunidade de usuários do OmegaT. No que tange a tarefas de rotina, é garantido aos voluntários que trabalham no projeto OmegaT o máximo de liberdade possível para determinar como contribuirão. Esta abordagem sem interferência é descrita pelo atual coordenador do projeto OmegaT como "anarquia delegada".

Em algumas áreas, é necessário algum grau de coordenação. Por exemplo, qualquer pessoa é livre para fazer alterações no código do OmegaT e enviar essas alterações para a equipe de desenvolvedores. Contudo, somente se tais alterações forem consideradas benéficas para os usuários do OmegaT, é que a equipe de desenvolvedores irá incluí-las no programa. Isto assegura que os interesses dos usuários sejam protegidos, e também resultou na obtenção do prêmio Garantia Softpedia "100% LIVRE" para o OmegaT.


Desenvolvimento do OmegaT

Como em todas as partes do projeto OmegaT, o trabalho de programação também é feito por voluntários, e, em consonância com a filosofia do projeto, os desenvolvimentos e melhorias feitos pelos voluntários estão, em última análise, à critério deles. No entanto, há margem considerável no projeto para os usuários sugerirem melhorias e influenciar o curso do desenvolvimento futuro.

A primeira delas é o grupo de usuários do OmegaT. Este é o foco principal da comunidade de usuários do OmegaT. Novos usuários do OmegaT são encorajados a fazer quaisquer sugestões aqui. Uma razão para isso é que a função desejada pode já ter sido implementada, ou pode haver razões do por que não pode ser implementada. Funções alternativas convenientes ou soluções também podem existir.

Detalhadas discussões técnicas sobre alterações são melhor aproveitadas se feitas diretamente com os desenvolvedores do OmegaT. Para este propósito, existe uma dedicada lista de discussão de desenvolvedores.

Depois que um usuário formulou uma proposta de alteração ou aprimoramento para o OmegaT, existe um procedimento definido para submete-la à consideração pela equipe de desenvolvedores. Isto permite que as inúmeras sugestões sejam gerenciadas mais facilmente. O procedimento é denominado como "RFE" (Request for Feature Enhancement, solicitação de melhoria de recurso) e é gerido aqui.

Literalmente centenas de RFE já foram feitos e muitos já foram implementados. No entanto, a submissão de um RFE não significa que o recurso ou melhoria desejada será automaticamente implementada. A aprovação ou não de tal solicitação depende, em primeiro lugar, da equipe de desenvolvedores considera-la benéfica para o OmegaT, e, em segundo lugar, se um dos desenvolvedores está disposto a trabalho na implementação do mesmo.

Os desenvolvedores do OmegaT, que são todos voluntários, podem estar mais inclinados a implementar um RFE se o trabalho de desenvolvimento for financiado pelo solicitante. O patrocínio do desenvolvimento já resultou na adição de determinados recursos. Caso você esteja disposto a financiar o trabalho de implementação de um recurso de seu interesse, entre em contato com a equipe de desenvolvedores, em particular, para uma cotação. Também pode ser possível que desenvolvimentos encomendados de desenvolvedores externos sejam implementados no OmegaT. No entanto, assim como acontece com os desenvolvimentos gratuitos, ainda se aplica o princípio de que quaisquer alterações para o OmegaT devam ser benéficas para o aplicativo e seus usuários.
