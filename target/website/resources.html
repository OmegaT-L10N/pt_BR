Outros recursos do OmegaT

Plugin de Tokenizers Fornece reconhecimento de formas flexionadas nas correspondências parciais e na busca de termos no glossário. As Instruções descrevendo os plugin de tokenizers também estão disponíveis.
Plugin da LanguageTool Verifica erros de linguagem (estilo, gramática etc).
Plugin de scripts Permite que os scripts externos acessem os recursos do OmegaT. (Não deve ser confundido com o recurso "exportar texto", que também permite que scripts sejam usados com o OmegaT, mas em um nível mais básico).
Plugin de filtros da Okapi Fornece filtros para lidar com mais formatos de arquivo, incluindo o Trados TTX.
Plugin da Apertium Integra o recurso de tradução automática da Apertium ao OmegaT.

Importante: para usar os plugins, você precisa instalar uma versão mais recente do OmegaT.

Utilitários

Utilitário alinhador "Aligner" Escrito em Python. Com a contribuição de Dmitri Gabinski.
Utilitário alinhador "Bligner" Escrito em Python/Perl. Com a contribuição de Didier Briel
TMXMerger Um script de linha de comando em linguagem Java para a fusão de dois ou mais arquivos TMX. Com a contribuição de Henry Pijffers.
TMXCleaner Um script de linha de comando em linguagem Java para a remoção de unidades de tradução "redundantes" de um arquivo TMX: exclui todas as UTs sem segmento de chegada ou em que o segmento de chegada seja idêntico ao segmento de partida. Com a contribuição de Henry Pijffers.
Toxic Um utilitário para a conversão de arquivos Trados TagEditor (TTX) para um formato intermediário, permitindo que sejam traduzidos no OmegaT. Esta coleção de scripts pode ser útil para programadores com conhecimento de tcl/tk (Tool Command Language). Aconselha-se que os outros usuários utilizem o Plugin de Filtros da Okapi, em vez do Toxic. Com a contribuição de Marc Prior.
Toxic para Windows Versão do Toxic executável no Windows.
Toxic para Linux Versão binária do Toxic para Linux.
Scripts do Toxic Versões de scripts do Toxic. Independente de plataforma; requer tcl/tk.
Stripper Um script para extrair texto para tradução a partir de arquivos de texto sem formatação (e.g.: código fonte de computador) para traduzir no OmegaT. Requer tcl/tk. Com a contribuição de Marc Prior.
TKount Utilitário para contagem de texto. Requer tcl/tk, Abiword e pdftotext. Foi testado no Linux. Com a contribuição de Marc Prior.
Ondiloto ONline DIctionary LOokup TOol (ferramenta de pesquisa em dicionário on-line). Requer Linux, w3m e tcl/tk (ambos geralmente fornecidos com o Linux). Com a contribuição de Marc Prior.
Priorview Utilitário tcl/tk que fornece uma pré-visualização do layout formatado de arquivos traduzidos. Com a contribuição de Marc Prior.
Wordfast TMX file converter Converts translation memories in TMX format generated by Wordfast to a form which can be read by OmegaT. Com a contribuição de Dmitri Gabinski.

Samuel Murray's scripts and procedures Descriptions of and links to Samuel Murray's scripts/procedures. Most of these scripts require MS Windows, although some procedures may also work on other platforms.
Scripts/procedures are available to enable you to do the following:
  • Skip segments containing a certain set of characters
  • Extract all source text from the project files into a single, large text file
  • Create a TMX file containing all strings, with identical source and target
  • Add terms to a glossary whilst translating in OmegaT
  • Add numbers to all segments, making them all "unique"
  • Remove all text except tags, or all tags except text
  • Check spelling (various procedures)
